# 更新日志

所有重要的更改都将记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-04-01

### ✨ 新增功能

#### 核心功能
- 🏠 **首页仪表盘** - 数据概览、中国地图分布、历史成就时间线
- 🌉 **桥梁博览** - 31座古桥知识库，支持类型/朝代/省份筛选
- 📄 **桥梁详情** - 完整信息展示、图片画廊、技术特点
- ⏳ **时间轴** - 按朝代展示桥梁发展历程

#### AI功能
- 🤖 **AI智能问答** - 基于GLM-4大模型的智能对话系统
- 🔍 **RAG知识库** - 桥梁专业知识检索增强生成

#### 可视化功能
- 🔬 **3D结构剖析** - Three.js构建赵州桥3D交互模型
- 📊 **数据统计** - 多维度数据分析可视化（饼图、柱状图、雷达图）
- ⚖️ **桥梁对比** - 2-3座桥梁数据对比功能

#### 互动功能
- 🏗️ **古桥建造坊** - 拖拽组件建造模拟器，实时评分
- 🏆 **知识挑战赛** - 答题闯关，徽章成就系统
- ❤️ **收藏功能** - 本地存储收藏桥梁

#### 内容功能
- 👨‍🔧 **匠心人物** - 建造者传记、工匠精神解读
- ℹ️ **关于项目** - 项目介绍、AI协同说明

### 🎨 UI/UX

- 古风UI设计：朱红+金色+米白配色
- 卷轴标题、印章标签、装饰角元素
- 完整响应式适配（桌面/平板/手机）
- 暗色/亮色主题支持

### 🛠️ 技术架构

- 前端：Vue 3 + Element Plus + ECharts + Three.js
- 后端：Flask + SQLite
- AI：GLM-4 + RAG
- 部署：Docker + Nginx

### 📦 部署支持

- Docker一键部署脚本（Linux/macOS/Windows）
- docker-compose容器编排
- GitHub Actions自动构建
- 完整部署文档
