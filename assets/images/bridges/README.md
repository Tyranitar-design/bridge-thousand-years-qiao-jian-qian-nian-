# 桥梁图片素材

本目录用于存放桥梁相关图片素材。

## 图片命名规范

```
{桥梁ID}-{序号}.jpg

例如：
zhaozhou-bridge-1.jpg  # 赵州桥全景
zhaozhou-bridge-2.jpg  # 赵州桥近景
lugou-bridge-1.jpg     # 卢沟桥全景
```

## 图片要求

- **格式**：JPG / PNG
- **分辨率**：建议 1920x1080 或更高
- **文件大小**：单张不超过 2MB
- **版权**：确保图片可商用或已获得授权

## 图片来源建议

1. **官方渠道**
   - 各地文物局官网
   - 景区官方网站
   - 国家博物馆数字藏品

2. **免费图库**
   - Unsplash (unsplash.com)
   - Pexels (pexels.com)
   - Pixabay (pixabay.com)

3. **自行拍摄**
   - 实地考察拍摄
   - 注意保护文物
