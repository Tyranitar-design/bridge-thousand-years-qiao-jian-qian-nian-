# GLM-4 API Key 配置指南

## 📝 获取 API Key 步骤

### 第一步：注册智谱AI账号

1. 访问 **https://open.bigmodel.cn/**
2. 点击右上角「注册/登录」
3. 可以使用手机号或微信登录

### 第二步：获取 API Key

1. 登录后进入「控制台」
2. 点击左侧菜单「API Keys」
3. 点击「创建新的API Key」
4. 复制生成的 API Key（只显示一次，请保存好）

### 第三步：配置项目

打开文件 `backend/.env`，将 API Key 填入：

```env
GLM_API_KEY=你的API_Key
```

**示例**：
```env
GLM_API_KEY=abc123def456ghi789.jkl012mno345pqr
```

### 第四步：重启后端服务

```powershell
cd backend
py -3 run.py
```

## ✅ 验证配置

配置完成后，访问前端 AI 问答页面，发送问题测试：

**测试问题**：
- "赵州桥为什么能屹立1400年不倒？"
- "中国古代桥梁有哪些主要类型？"

如果返回正常的AI回答（而非本地知识库），说明配置成功！

## ⚠️ 注意事项

1. **API Key 安全**：不要将 `.env` 文件提交到 Git
2. **额度监控**：定期查看控制台的使用量
3. **错误排查**：如果提示 API 错误，检查 Key 是否正确

---

## 🔧 常见问题

### Q: 提示 "API Key 无效"
A: 检查 Key 是否完整复制，没有多余空格

### Q: 提示 "余额不足"
A: 登录控制台查看账户余额，必要时充值

### Q: 返回的是本地知识库内容
A: 检查 `.env` 文件是否在 `backend` 目录下，并重启服务
