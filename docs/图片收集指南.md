# 桥梁图片下载指南

## 📥 手动下载方法

### 方法一：百度百科

1. 打开浏览器，访问 https://baike.baidu.com
2. 搜索桥梁名称（如"赵州桥"）
3. 点击进入词条页面
4. 找到图片，右键 → "在新标签页打开图片"
5. 保存图片到本地

### 方法二：必应图片搜索

1. 访问 https://cn.bing.com/images
2. 搜索桥梁名称
3. 选择合适的图片，点击查看原图
4. 右键保存

### 方法三：搜狗图片

1. 访问 https://pic.sogou.com
2. 搜索桥梁名称
3. 选择大尺寸图片
4. 右键保存

## 📐 图片处理要求

下载后需要：
1. **调整尺寸**：800 × 450 像素（可用画图工具或在线工具）
2. **压缩大小**：< 500KB
3. **重命名**：按下方命名规则

### 在线图片处理工具
- https://www.iloveimg.com/zh-cn（压缩图片）
- https://www.gaitubao.com/（改图宝）
