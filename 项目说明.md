# 【桥见千年】中国古代桥梁智慧可视化平台

## 项目说明

【桥见千年】是一个专注于中国古代桥梁文化传承与知识普及的数字化平台。项目以1911年前的中国古代桥梁为研究对象，收录31座著名古桥的完整数据，涵盖赵州桥、卢沟桥、洛阳桥等经典案例，每座桥梁均包含基本信息、技术特点、建造者传记、历史文献及诗词典故等多维度内容。

平台采用Vue 3 + Flask技术架构，融合Three.js 3D模型展示、ECharts数据可视化、GLM-4大模型智能问答等现代技术，打造了包括桥梁博览、时间轴演进、结构剖析、古桥建造坊、桥梁对比、知识挑战赛在内的12个功能模块。其中，"古桥建造坊"通过拖拽式建造模拟让用户体验古桥搭建过程，"知识挑战赛"以答题闯关形式普及桥梁知识，"AI问答"则基于GLM-4大模型实现智能对话，为用户提供个性化的知识解答。

项目采用AI协同开发模式，AI作为开发伙伴参与架构设计、代码编写、文档生成等全流程工作，AI工具使用占比约85%。界面设计采用朱红、金色、米白为主色调的古风UI，并实现桌面端、平板端、手机端的响应式适配，让千年桥梁智慧以更生动、更易触达的方式走进大众视野。
